void insert(int index, Student s) {
  // 元素后移: 需移动 N-i 个元素
  for(int i=size; i>index; i--)
    array[i] = array[i-1];
}

内存特征：物理连续存放，插入需大量数据搬移。

单链表 (Linked List)

O(1) 插入效率

void insert(Node* prev, Student s) {
  // 指针重定向: 仅修改两个链接
  Node* newNode = new Node(s);
  newNode->next = prev->next;
  prev->next = newNode;
}

内存特征：离散存放，指针关联，扩容成本极低。
